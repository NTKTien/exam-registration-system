ĐỒ ÁN THỰC HÀNH - PTTK HTTT - HK2 2425 

Trung tâm tổ chức thi chứng chỉ anh ngữ và tin học ACCI chuyên cung cấp các chương trình 
hỗ trợ khảo sát tiếng Anh và tin học cho các cấp độ từ tiều học đến Đại học. 
Đồ án này yêu cầu cài đặt hệ thống hỗ trợ quy trình tổ chức thi chứng chỉ của trung tâm trên.

Cụ thể: 

  1. Phân tích nghiệp vụ
     - Xây dựng Use case nghiệp vụ
     - Đặt tả usecase
     - Thiết kế Activity diagram
     - Object diagram
  2. Phân tích hệ thống
     - Use case hệ thống
     - Đặt tả use case hệ thống
  2. Xác định chức năng tự động hóa của hệ thống
  3. Đề xuất giải pháp thiết kế phù hợp
     - CSDL QH
     - Màn hình giao diện
     - Sơ đồ lớp mức phân tích
     - Sequence
  5. Cài đặt hệ thống sau thiết kế
     - Mỗi thành viên phụ trách 1 giao diện
     - Hệ thống được cài đặt bằng WinForms trên môi trường Visual Studio
